# Katalon API & Kafka Test Project

## Features
- REST API Testing (Producer & Consumer)
- Kafka Consumer Testing (Local Docker)

## Tech Stack
- Katalon Studio
- Apache Kafka (Docker)
- Groovy

## How To Run
1. Install Docker & Java
2. docker compose up -d
3. Create topic test-topic
4. Open project in Katalon
5. Run test cases
